1) Игроки делятся на команды. Каждая команда выбирает одного члена, который будет объяснять слова, и одного члена, который будет угадывать.
2) Каждая команда выбирает карту со словами из колоды. Описание слова на карте должно быть объяснено без использования самого слова, его частей или родственных слов.
3) Время для угадывания каждого слова ограничено. Если слово не угадано за отведенное время, команда должна передать ход другой команде.
4) За каждое угаданное слово команда получает очко. Побеждает команда, которая первой наберет определенное количество очков (обычно 10-15).
5) Запрещено использовать жесты, показывать или указывать на предметы, а также использовать звуки, которые напоминают звучание самого слова.
6) Игра продолжается до тех пор, пока не будет набрано необходимое количество очков или пока все слова на картах не будут объяснены.
